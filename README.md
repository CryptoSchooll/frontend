# CryptoSchool's Frontend Repository

## Stack
- **Bun**
- **React**
- **Typescript**
- **Vite**
- **TailwindCSS**
- **Bun**

## Getting Started

1. Clone the repository:
```bash
git clone git@github.com:CryptoSchooll/frontend.git
```

2. Navigate to the project directory:
```bash
cd frontend
```

3. Install dependencies:
```bash
bun install
```

4. Start the dev server:
```bash
bun run dev
```
